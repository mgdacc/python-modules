¿Qué es un Módulo?

Un archivo .py que contiene funciones, clases y variables que puedes usar en otros archivos.

2 / 13

¿Por qué usar Módulos?

Para organizar el código, reutilizar funciones y acceder a la enorme librería estándar de Python.

3 / 13

Importar un Módulo: import

La forma más básica de usar un módulo. Se accede al contenido con nombre_modulo.funcion().

import math

raiz_cuadrada = math.sqrt(25)
print(raiz_cuadrada) # 5.0
4 / 13

Módulo Built-in: math

Proporciona funciones y constantes matemáticas.

import math

print(math.pi)      # 3.14159...
print(math.pow(2, 3)) # 8.0 (2 elevado a 3)
5 / 13

Módulo Built-in: random

Permite generar números aleatorios y realizar selecciones al azar.

import random

numero_aleatorio = random.randint(1, 10)
print(numero_aleatorio) # Un número entre 1 y 10

frutas = ["manzana", "banana", "cereza"]
fruta_elegida = random.choice(frutas)

Módulo Built-in: datetime

Para trabajar con fechas y horas.

import datetime

ahora = datetime.datetime.now()
print(ahora.year)
print(ahora.strftime("%Y-%m-%d %H:%M:%S"))
7 / 13

Importar con un Alias: as

Permite dar un nombre más corto a un módulo al importarlo.

import datetime as dt

ahora = dt.datetime.now()
print(ahora.year)
8 / 13

Importar Específicos: from

Importa solo una función o variable de un módulo. No necesita prefijo para usarse.

from math import sqrt, pi

print(sqrt(25)) # 5.0 (no se usa math.sqrt)
print(pi)       # 3.14159...
9 / 13

Importar Todo: * (No recomendado)

Importa todo el contenido de un módulo. Puede causar conflictos de nombres y no es claro.

from math import *

# Se puede usar todo sin prefijo
print(sqrt(25))
print(pi)

Módulos Custom (Propios)

Cualquier archivo .py puede ser un módulo.

# Archivo: mis_funciones.py
def saludar(nombre):
    return f"Hola, {nombre}"

Usar un Módulo Custom

Se importa igual que un módulo built-in, siempre que esté en la misma carpeta.

# Archivo: principal.py
import mis_funciones

mensaje = mis_funciones.saludar("Ana")
print(mensaje)

Paquetes (Packages)

Un paquete es una carpeta que contiene múltiples módulos, organizando proyectos más grandes.
